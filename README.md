# Roblox NotificationBot UI (dev)

Small Vite + React dev setup to run the UI locally.

Prerequisites:
- Node.js (16+ recommended)

Install & run:

```bash
npm install
npm run dev
```

Open the URL printed by Vite (usually `http://localhost:5173`). Tailwind classes are enabled via the CDN for quick development.
